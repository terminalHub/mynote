## 3 ==JS使用方式==

### 3.1 内容概述

```javascript
(1)行内式：
	在当前元素上直接编写JS代码  
	了解
	<input type="button" value="普通按钮" onclick="alert('行内式')">
        
(2)嵌入式：
    在当前HTML文件中添加script标签,编写JS代码
	script标签位置不固定,比如在head标签中 或者 在body标签下面
    <script>
        console.log(111);
    </script>

(3)外链式：
    单独创建一个JS文件,在JS文件中编写JS代码,最后在HTML文件中引入JS文件
	<script src="../js/myjs.js"></script>
```

### 5.4 常用事件

#### 5.4.1 ==单击事件==

##### 5.4.1.1 内容概述

```
鼠标单击时，发生的事件  onclick

操作标签体内容：    获取值：元素.innerHTML       设置值：元素.innerHTML=值
操作value属性值：  获取值：元素.value            设置值：元素.value=值
操作元素样式值：    获取值：元素.style.属性名称    设置值：元素.style.属性名称=值
```

##### 5.4.1.2 案例

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 50px;
            height: 50px;
            border: 1px solid black;
        }
    </style>
</head>

<body>
    <p id="spring">春暖花开</p>
    <input id="userName" type="text" value="tom">
    <br>
    <br>
    <div id="myDiv">

    </div>
    <br>
    <br>
    <input type="button" value="点击" onclick="btnFun()">
</body>
<script>
    // 单击事件:onclick

    // 定义函数
    function btnFun() {
        // 1.获取 标签体文本内容  找元素 获取内容
        // 获取p元素
        var pEle = document.getElementById('spring');
        // 获取文本内容
        console.log(pEle.innerHTML);
        // 2.修改 标签体文本内容  找元素 修改内容
        pEle.innerHTML = '春意盎然';
        // 3.获取 value属性值   找元素 获取value属性值
        var inputEle = document.getElementById('userName');
        // 获取value属性值
        console.log(inputEle.value);
        // 4.设置 value属性值   找元素 设置value属性值
        inputEle.value = 'jack';
        // 5.设置 样式  设置div背景颜色
        // 获取div元素
        var divEle = document.getElementById('myDiv');
        // 设置样式
        divEle.style.backgroundColor = 'green';
        // 获取样式
        console.log(divEle.style.backgroundColor);
    }
</script>

</html>
```

##### 5.4.3.2 案例

```html
    // 获取input元素
    var inputEle = document.getElementById('myText');
    // 1.获取到焦点时,设置文本颜色
    inputEle.onfocus = function () {
        inputEle.style.color = 'pink';

    // 2.失去焦点时,设置背景颜色
    inputEle.onblur = function () {
        inputEle.style.backgroundColor = 'yellow';
    }
</script>

</html>
```

## 6 BOM

### 6.1 内容概述

```
(1)Browser Object Model，浏览器对象模型
(2)把浏览器看作一个对象
(3)BOM包含一些对象，每个对象都有属性和方法，和浏览器窗口交互操作
(4)核心对象：window对象，是JS访问浏览器的接口；全局对象，定义在全局作用域中的变量和函数都会变成window对象的属性和方法
(5)window对象：history对象、location对象、document对象等    ***
(6)浏览器厂商制定的接口，兼容性不好
```

## 9 ==JSON==

### 9.1 内容概述

```
(1)JavaScript Object Notation，JS的对象表示形式
(2)轻量级数据交换格式
(3)语法：
	{key:val,key:val,...}
	标准写法：{"key":val,"key":val,...}   val：String类型加引号;number类型或boolean类型不加引号   推荐写法
	简化写法：{key:val,key:val,...}       val：String类型加引号;number类型或boolean类型不加引号   纯前端	
(4)常见写法：
	数组和JSON嵌套使用
	获取值：
		数组   使用索引
		JSON  通过key获取value
(5)常用方法：
	遍历数组：数组.forEach(function(data,index,array))  data：遍历到的每项数据  index：索引值  array：数组
	将JSON格式字符串转成JSON对象：JSON.parse()
	将JSON对象转成字符串：JSON.stringify()
```

### 9.3 省市区案例

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!-- 引入js文件 -->
    <script src="../js/area.js"></script>
</head>

<body>
    <!-- 三个下拉选  省市区 -->
    <select id="province" onchange="queryCity()">
        <option value="" disabled selected>-----请选择-----</option>
    </select>省
    <select id="city" onchange="queryCounty()">
        <option value="" disabled selected>-----请选择-----</option>
    </select>市
    <select id="county">
        <option value="" disabled selected>-----请选择-----</option>
    </select>区县
</body>
<script>
    // 获取省元素
    var prvinceEle = document.getElementById('province');
    // 获取市元素
    var cityEle = document.getElementById('city');
    var cityData = [];
    // 获取区县元素
    var countyEle = document.getElementById('county');


    // 加载省数据  进入页面加载省数据
    var proOption = '';
    areaData.forEach(function (province, index) {
        // console.log(province);//每项省数据
        proOption += '<option value="' + province.code + '">' + province.name + '</option>';
    });
    prvinceEle.innerHTML += proOption;


    // 加载市数据  省数据改变时加载市数据
    function queryCity() {
        // 把市数据设置成 请选择
        cityEle.innerHTML = '<option value="" disabled selected>-----请选择-----</option>';
        // 把区县数据设置成 请选择
        countyEle.innerHTML = '<option value="" disabled selected>-----请选择-----</option>';
        // 所选省数据value值
        // console.log(prvinceEle.value);
        var proCode = prvinceEle.value;
        areaData.forEach(function (province) {
            // 定位到省数据
            if (proCode == province.code) {
                cityData = province.city;
                var cityOption = '';
                // 遍历市数据
                province.city.forEach(function (city) {
                    cityOption += '<option value="' + city.code + '">' + city.name + '</option>';
                })
                cityEle.innerHTML += cityOption;
            }
        })
    }

    // 加载区县数据  市数据改变时，加载区县数据
    function queryCounty() {
        // 把区县数据设置成 请选择
        countyEle.innerHTML = '<option value="" disabled selected>-----请选择-----</option>';
        // 所选市数据 code
        // console.log(cityEle.value);
        var cityCode = cityEle.value;
        // 遍历市数据 
        cityData.forEach(function (city) {
            // 匹配市数据
            if (cityCode == city.code) {
                var countyOption = '';
                // 遍历区县数据
                city.area.forEach(function (county) {
                    countyOption += "<option value='" + county.code + "'>" + county.name + "</option>";
                })
                countyEle.innerHTML += countyOption;
            }
        })
    }

</script>

</html>
```
